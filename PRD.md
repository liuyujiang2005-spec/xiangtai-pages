## 达人短视频分发 APP - 产品需求文档（PRD）

### 1. 项目背景与目标

- **背景**：品牌/小老板希望通过达人在抖音、小红书等平台发布带货短视频，提高曝光与转化，但缺少统一的素材分发、任务管理与结算平台。
- **目标**：
  - 为管理员提供一套统一的平台，用于管理短视频素材、配置任务、审核达人投稿、进行风控与结算。
  - 为达人提供领取任务、下载视频、发布作品并反馈链接、查看积分和结算的闭环体验。
  - 为客户提供查看投放效果、管理样品寄送与任务需求的入口。

### 2. 角色与端口

- **管理员端（Admin Portal）**
  - 管理短视频素材（含云盘链接）。
  - 配置/发布任务，指定积分、要求、是否露脸等。
  - 审核达人投稿，查看 AI/风控结果并做最终决策。
  - 管理积分与结算，对接打款流程。
  - 查看全局统计报表与审计日志。

- **达人端（Influencer Portal）**
  - 浏览并领取任务（按平台、类型、积分筛选）。
  - 下载视频素材（通过云盘或平台内部下载链接）。
  - 完成发布后，反馈作品链接和必要说明。
  - 查看当前积分、本周预计结算与历史任务记录。
  - 接收违规/扣分/锁定期内删除等风控提示。

- **客户端（Client Portal）**
  - 提交任务/产品需求，与平台合作并寄送样品。
  - 绑定长期合作达人池（尤其是露脸达人）。
  - 查看达人发布作品及整体投放效果（数量、基础数据等）。

### 3. 核心功能设计

#### 3.1 视频存储与分发

- **素材来源**
  - 管理员在外部云盘（谷歌云盘、百度网盘、OSS 等）上传原始带货短视频。
  - 在管理员端为每条视频录入：
    - 标题
    - 类型：露脸（face）/ 讲解（explain）
    - 云盘下载链接 / 内部存储链接
    - 适合平台（抖音 / 小红书 / 快手 等）
    - 备注：人设建议、文案建议、是否需要投流。

- **在 APP 中的呈现（达人端）**
  - 任务列表以卡片形式展示视频：
    - 封面（如有）、标题、类型（露脸/讲解）标签。
    - 云盘链接或一键“打开云盘/复制链接”按钮。
    - 推荐平台、任务要求与可获得积分。
  - 达人领取任务后，可在「我的任务」中再次查看该视频及要求。

#### 3.2 任务与积分闭环

- **任务生命周期**
  1. 管理员创建任务：
     - 关联视频素材
     - 设置任务类型（露脸/讲解）、目标平台、可领取数量（可选）
     - 设置积分奖励（如 10 分/条）
  2. 达人浏览并领取任务：
     - 系统记录任务领取（含日期、达人 ID、任务 ID）
     - 可配置每日领取上限（如 10 条/日）
  3. 达人下载素材并发布：
     - 使用云盘链接下载原视频
     - 按要求剪辑/添加文案，在对应平台发布
  4. 达人回填作品链接：
     - 在达人端提交「作品链接」及可选补充说明
     - 状态变为「待审核」
  5. 管理员审核：
     - 通过 → 系统自动增加达人积分，并记录积分流水
     - 驳回 → 可要求重新发布，记录驳回原因
  6. 锁定期风控（防删）：
     - 在锁定期（如 3/5 天）内监控作品是否被删或异常
     - 如违约可按规则扣除积分/限制后续任务

- **积分与结算**
  - 积分获取：每条通过审核的投稿，根据任务配置获得固定积分。
  - 积分与现金兑换比例：1 积分 = 1 泰铢（可在后台配置）。
  - 结算周期：以「周」为周期：
    - 每周统计上周被认定为“有效”的积分总数
    - 管理员导出结算报表并线下或通过支付渠道打款
    - 可在系统中标记结算状态（未结算 / 已结算 / 异常）

#### 3.3 防删机制与审核流程

- **AI/自动巡检（防删机制）**
  - 对达人提交的作品链接执行定时巡检：
    - 检查链接是否仍可访问（未被平台删除/设为私密）。
    - 可扩展：根据标题/文案/封面，对明显违规内容进行初步打标。
  - 巡检结果写入数据库：
    - 正常 / 疑似删除 / 违规风险
  - 告警策略：
    - 对于疑似删除或无法访问的作品：
      - 在管理员后台高亮提示
      - 发送站内通知（后续可集成邮件/企业微信）
      - 阻止该条作品自动结算，需人工复核

- **时间锁定与扣分规则**
  - 每条通过审核的作品进入锁定期（如 3 天或 5 天，可配置）：
    - 若在锁定期内被删除/设为私密：
      - 系统标记为违约
      - 执行扣分：全额扣除该条任务所得积分或按比例扣减（可配置）
    - 对于多次违约达人：
      - 可降低其等级，限制每日可领取任务数，或暂时封禁任务领取

- **人工审核流程**
  - 审核入口：
    - 管理员端「投稿审核」页面按列表展示：
      - 达人、任务、作品链接、提交时间、AI 检测结果
  - 审核步骤：
    - 人工打开链接检查内容：
      - 是否与任务视频对应（是否盗用他人素材）
      - 是否带货文案/话术符合要求
      - 是否存在平台高危行为/敏感内容
    - 审核结果：
      - 通过：
        - 更新投稿状态为「已通过」
        - 自动为达人增加积分，并记录积分流水
        - 启动锁定期计时，用于后续防删监控
      - 驳回：
        - 更新状态为「已驳回」
        - 记录驳回原因（展示给达人）
        - 不产生积分

### 4. APP 端口设置与达人匹配

#### 4.1 客户端（Client）

- 功能范围：
  - 提交合作意向与任务需求（产品信息、目标平台、预算范围、是否露脸）。
  - 维护与公司的合作信息及样品寄送情况。
  - 查看其任务的执行情况：
    - 有哪些达人参与
    - 每条作品链接列表
    - 基础表现数据（如后续接入）：
      - 播放量、点赞、评论、收藏等。

#### 4.2 管理员端（Admin）

- 核心模块：
  - **素材管理**：视频录入、类型标记（露脸/讲解）、云盘链接维护、上下架。
  - **任务管理**：创建任务、设置积分、限制条件（平台、人设要求、是否露脸）。
  - **达人管理**：
    - 审核达人资料
    - 打标签：露脸达人 / 讲解达人 / 平台专长（抖音/小红书等）
    - 设置等级与黑名单。
  - **审核与风控**：投稿审核列表、AI 风险结果展示、防删异常告警。
  - **积分与结算**：积分汇总、积分流水、结算导出与打款记录。
  - **审计与统计**：登录/操作审计日志、任务/投稿/下载统计报表。

#### 4.3 达人端（Influencer）

- 核心体验：
  - 任务大厅：
    - 按平台、类型（露脸/讲解）、积分排序和筛选任务。
    - 支持查看任务详情和视频说明。
  - 我的任务：
    - 显示已领取任务列表及当前状态（未发布 / 待审核 / 已通过 / 被驳回 / 锁定期中 / 已结算）。
  - 投稿与反馈：
    - 为每条任务提交作品链接（可多平台多条时按规则配置）。
    - 可查看审核意见与驳回原因。
  - 积分与收益：
    - 当前可用积分、本周预计结算积分、历史收益统计。
    - 查看每条任务对应的积分流水记录（方便对账）。

### 5. 达人匹配策略（尤其是露脸需求）

- **达人标签体系**
  - 基础标签：
    - 是否愿意露脸：是 / 否
    - 人设类型：宝妈 / 职场 / 学生 / 户外 / 美妆 / 运动等
    - 主攻平台：抖音 / 小红书 / 快手 / B 站等
  - 业务标签：
    - 是否为长期合作达人
    - 历史表现：完成率、审核通过率、违规次数

- **任务配置中的匹配条件**
  - 当任务标记为「必须露脸」：
    - 默认只对「露脸达人 = 是」开放领取。
    - 可额外限定人设类型（如宝妈/美妆）。
  - 对于有长期露脸需求的客户：
    - 管理员可维护「客户 – 指定达人池」：
      - 该客户新建的露脸任务，仅向其绑定的达人池开放。

- **前台呈现与权限控制**
  - 达人端任务列表：
    - 对不符合条件的达人隐身或仅展示但不可领取（提示“仅限特邀达人/露脸达人”）。
  - 管理端：
    - 可在任务详情中查看当前匹配达人池规模与覆盖度。

### 6. 非功能性与后续规划（摘要）

- **安全与风控**
  - JWT + Refresh Token + 角色鉴权 + 细粒度 admin 权限。
  - 防刷：
    - 登录/领取/下载限流
    - 下载与领取记录日志，用于风控分析。
  - 防删机制：
    - 锁定期 + AI/脚本巡检 + 违约扣分。

- **可观测性**
  - request-id 贯穿所有请求，写入审计日志，便于排障。
  - 管理端提供审计查询与 CSV 导出，为后续 BI/告警打基础。

- **后续可扩展方向**
  - 将当前 SQLite/sql.js 持久化方案升级为 Postgres/MySQL + 迁移工具。
  - 接入真实 AI 内容安全/相似度识别，自动识别盗用与违规内容。
  - 丰富客户端报表（转化漏斗、达人排行榜、平台分布等）。
